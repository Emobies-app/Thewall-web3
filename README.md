### 🛠️ Tech Stack

- **Frontend**: Next.js 14 + TypeScript + Tailwind
- **Wallet**: Reown (WalletConnect) + Ethers Adapter
- **Blockchain**: Alchemy (RPC + Gas Manager) + Helius (Solana)
- **AI**: Claude (Anthropic) via Emowall AI
- **Charts**: CoinGecko
- **Deployment**: Vercel
