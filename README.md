## 🚀 Deploy

```bash
# Clone
git clone https://github.com/Emobies-app/Thewall-web3

# Install
npm install

# Environment variables
ALCHEMY_API_KEY=
ALCHEMY_GAS_POLICY_ID=
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=
HELIUS_API_KEY=
ETHERSCAN_API_KEY=
WALL_CLOUDE_AI=
NILEDB_POSTGRES_URL=

# Run
npm run dev
```
